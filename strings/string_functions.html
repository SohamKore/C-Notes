String Functions

Include: #include <string.h>

This library includes the string functions described below.

1. strupr(string)

Converts the string into uppercase.

strupr(str);
puts(str);

2. strlwr(string)

Converts the string into lowercase.

strlwr(str);
puts(str);

3. strlen(string)

Returns the string length/size.

char str[] = "abc";
int x = strlen(str);
printf("%d", x);

4. strcpy(destination, source)

Copies the content of the source string into the destination string.

char str1[] = "";
char str2[] = "Hello";
strcpy(str1, str2);
puts(str1);

5. strcat(first, second)

Concatenates/combine the second string into the first string.

char str1[] = "Hello";
char str2[] = "World";
strcat(strcat(str1, " "), str2);
puts(str1);

6. strcmp(first, second)

Compares the first string with the second string. If both strings are the same, it returns 0.

char str1[] = "helxlo";
char str2[] = "hellasdo";
int result = strcmp(str1, str2);
printf("%d", result);

7. strrev(string)

Reverses the string.

char str[] = "Hello";
strrev(str);
puts(str);
