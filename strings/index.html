String Functions in C

1. strupr()

Description: Converts a string to uppercase.

Syntax: char* strupr(char* str);

Returns: A pointer to the converted string.

char str[] = "hello world";
strupr(str);
puts(str); // Output: "HELLO WORLD"

2. strlwr()

Description: Converts a string to lowercase.

Syntax: char* strlwr(char* str);

Returns: A pointer to the converted string.

char str[] = "HELLO WORLD";
strlwr(str);
puts(str); // Output: "hello world"

3. strlen()

Description: Returns the length of a string.

Syntax: size_t strlen(const char* str);

Returns: The length of the string, excluding the null terminator.

char str[] = "hello";
size_t len = strlen(str);
printf("%zu", len); // Output: 5

4. strcpy()

Description: Copies a source string into a destination string.

Syntax: char* strcpy(char* dest, const char* src);

Returns: A pointer to the destination string.

char dest[20];
char src[] = "hello";
strcpy(dest, src);
puts(dest); // Output: "hello"

5. strcat()

Description: Concatenates the second string to the first string.

Syntax: char* strcat(char* dest, const char* src);

Returns: A pointer to the concatenated string.

char dest[20] = "hello";
char src[] = " world";
strcat(dest, src);
puts(dest); // Output: "hello world"

6. strcmp()

Description: Compares two strings.

Syntax: int strcmp(const char* str1, const char* str2);

Returns:

  • 0 if strings are equal.
  • A negative value if str1 is less than str2.
  • A positive value if str1 is greater than str2.

char str1[] = "hello";
char str2[] = "world";
int result = strcmp(str1, str2);
printf("%d", result); // Output depends on comparison

7. strrev()

Description: Reverses a string.

Syntax: char* strrev(char* str);

Returns: A pointer to the reversed string.

char str[] = "hello";
strrev(str);
puts(str); // Output: "olleh"

8. strchr()

Description: Finds the first occurrence of a character in a string.

Syntax: char* strchr(const char* str, int c);

Returns: A pointer to the first occurrence of the character, or NULL if not found.

char str[] = "hello";
char* ptr = strchr(str, 'e');
printf("%s", ptr); // Output: "ello"

9. strrchr()

Description: Finds the last occurrence of a character in a string.

Syntax: char* strrchr(const char* str, int c);

Returns: A pointer to the last occurrence of the character, or NULL if not found.

char str[] = "hello";
char* ptr = strrchr(str, 'l');
printf("%s", ptr); // Output: "lo"

10. strncpy()

Description: Copies up to a specified number of characters from the source string to the destination string.

Syntax: char* strncpy(char* dest, const char* src, size_t n);

Returns: A pointer to the destination string.

char dest[10];
char src[] = "hello";
strncpy(dest, src, 3);
dest[3] = '\0';
printf("%s", dest); // Output: "hel"
